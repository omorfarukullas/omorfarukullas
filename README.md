### MediconnectBD — Unified Smart Healthcare System

> **Bridging the healthcare access gap in Bangladesh through intelligent technology**

A full-stack platform connecting patients, doctors, hospitals, and emergency services — built with real-world scale in mind.

| Module | Description | Tech |
|--------|-------------|------|
| Appointment Engine | Booking + live queue tracking | Next.js · WebSockets |
| EMR System | Encrypted electronic medical records | PostgreSQL · Node.js |
| Resource Dashboard | Hospital inventory & bed management | REST API · MySQL |
| Emergency Dispatch | Real-time ambulance geolocation | Geo API · Express |
| Telemedicine | Video consultation platform | WebRTC · React |

`TypeScript` `Next.js` `Node.js` `PostgreSQL` `MySQL` `Docker` `REST API`
